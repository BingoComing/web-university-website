# Web技术大学官网项目文档

## 1. 项目概述

### 1.1 项目背景
为"Web技术大学"开发的官方网站，展示学校信息、新闻动态和校园活动，服务师生、校友和访客。

### 1.2 项目结构
- **首页**：展示校园要闻、学术讲座、通知公告
- **导航菜单**：包含学校概况、院系设置、新闻中心等栏目
- **内容页面**：媒体报道、图片新闻、视频集锦等
- **响应式设计**：适配移动端、平板和桌面设备

### 1.3 技术栈
- HTML5：页面结构
- CSS3：样式设计（Flexbox、CSS变量、媒体查询）
- JavaScript：交互功能（轮播图、下拉菜单）

## 2. 我的贡献

- **导航菜单优化**：调整主菜单项对齐和间距，优化下拉菜单定位，统一二级子菜单居中对齐
- **页面文本对齐**：统一首页和校园快讯页面的文本居中显示
- **响应式设计增强**：确保跨设备对齐一致性，优化移动端菜单样式

## 3. 核心技术实现

### 3.1 导航菜单设计

#### 3.1.1 主菜单项居中对齐
```css
.nav > ul > li {
  flex: 1; /* 均匀分配空间 */
  text-align: center;
}

.nav > ul > li > a {
  padding: 16px 20px;
  text-align: center;
}
```

#### 3.1.2 下拉菜单精确对齐
```css
.nav .dropdown {
  position: absolute;
  top: 100%; /* 紧贴一级菜单下方 */
  left: 0;
  width: 100%; /* 与一级菜单同宽 */
}
```

#### 3.1.3 二级子菜单居中
```css
.nav .dropdown li, .nav .dropdown li a {
  text-align: center;
}
```

### 3.2 页面文本居中

#### 3.2.1 模块标题居中
```css
h2 {
  text-align: center;
}

h2::after {
  left: 50%;
  transform: translateX(-50%); /* 装饰线居中 */
}
```

### 3.3 响应式设计
```css
@media (max-width: 992px) {
  .nav > ul { flex-direction: column; }
  /* 确保所有文本居中 */
  .nav > ul > li, .nav > ul > li > a, .nav .dropdown li a {
    text-align: center;
  }
}
```

## 4. 关键挑战与解决方案

| 挑战 | 解决方案 |
|------|----------|
| 跨设备对齐一致性 | 使用相对单位和Flexbox，媒体查询中明确设置文本对齐 |
| 文本居中与可读性平衡 | 采用"混合对齐"策略：标题居中，具体内容左对齐 |
| 下拉菜单定位准确性 | 使用`position: absolute`结合`top: 100%`和`width: 100%` |

## 5. 项目成果

- **视觉效果**：导航菜单整齐有序，跨设备显示稳定，文本居中增强视觉统一性
- **功能完整**：所有菜单项可点击，下拉菜单正常展开/折叠，响应式设计正常工作
- **代码质量**：结构清晰，使用CSS变量和模块化设计，符合现代CSS最佳实践
