# 合肥大学 Web 前端课程设计项目文档

## 1. 项目概述
本项目是基于 HTML5、CSS3 和 JavaScript 开发的合肥大学（Hefei University）校园门户网站。项目旨在展示学校概况、院系设置、新闻动态及招生就业等信息，同时应用了现代 Web 前端开发技术，实现了响应式布局、交互动效及表单验证等功能，以满足课程设计对技术深度和用户体验的要求。

## 3. 关键技术点

### 3.1 HTML5 语义化
- 使用了 `<header>`, `<nav>`, `<section>`, `<footer>` 等语义化标签，构建清晰的页面结构，提升 SEO 友好度和代码可读性。
- 使用 HTML5 表单控件（如 `type="email"`）配合 JS 进行前端验证。

### 3.2 CSS3 样式与布局
- **Flexbox 布局**：在导航栏、新闻列表、404 页面等模块广泛使用 Flexbox 实现灵活的居中和对齐。
- **CSS3 动画**：
  - **全局淡入 (Fade-in)**：通过 `@keyframes` 定义淡入效果，并应用到 `body`，实现页面加载时的平滑过渡。
  - **交互特效**：按钮悬停时的缩放 (`transform: scale`) 和颜色渐变 (`transition`)。
- **响应式设计**：虽未完全依赖框架，但通过 CSS 媒体查询和百分比布局，适配不同屏幕尺寸（如移动端菜单切换）。

### 3.3 JavaScript 交互逻辑
- **轮播图 (Carousel)**：原生 JS 实现首页 Banner 图片的自动轮播和手动切换。
- **表单验证**：在“联系我们”页面，对姓名、邮箱格式进行正则校验，并提供实时错误反馈。
- **搜索功能**：实现了简单的搜索关键词非空校验，并跳转至结果页（模拟搜索参数传递）。
- **404 倒计时重定向**：利用 `setInterval` 实现 5 秒倒计时，自动跳转回首页，提升用户体验闭环。
- **回到顶部**：监听滚动事件，动态显示“回到顶部”按钮，并实现平滑滚动效果。

## 4. 页面功能亮点
1.  **容错性设计**：新增 `404.html` 页面，当用户访问不存在的链接时，提供友好的提示并引导返回首页。
2.  **视觉体验**：统一的蓝色调（合大蓝）风格，配合阴影 (`box-shadow`) 和圆角设计，界面现代简洁。
3.  **完整性**：涵盖了学校官网常见的几乎所有栏目（概况、院系、新闻、招生、就业、校友等），页面数量丰富，链接逻辑通顺。

## 5. 总结
本项目综合运用了 Web 前端核心技术，不仅完成了静态页面的展示，还加入了丰富的动态交互和细节优化（如 404 页面、全局动画），充分体现了对用户体验的关注。代码结构清晰，易于维护和扩展。
